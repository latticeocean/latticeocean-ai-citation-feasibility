# What LatticeOcean Is

LatticeOcean is an **AI Citation Feasibility platform**.

It measures whether a webpage is structurally eligible to be cited inside AI-generated answers from systems such as ChatGPT, Gemini, and Perplexity.

AI answer engines do not rank pages the way traditional search engines do. Instead, they select documents that match the structural patterns of the answer being generated.

LatticeOcean analyzes these structural patterns and determines whether a page qualifies for citation.

---

# What AI Citation Means

An **AI citation** occurs when an AI answer engine references a webpage as a source while generating a response.

For example, when users ask questions such as:

• best CRM software  
• best analytics platforms  
• best observability tools  
• alternatives to a specific SaaS product  

AI systems often generate answers that reference external sources.

These cited sources influence buyer decisions because users frequently rely on AI answers during vendor evaluation.

If a website is not structurally compatible with the answer format, it will typically not appear in the citation list.

---

# How AI Answer Engines Select Sources

Traditional search engines rank webpages using signals such as backlinks, keyword relevance, and domain authority.

AI answer engines retrieve documents that match the **expected structure of the response**.

Common structural characteristics of cited documents include:

• clear headings aligned with the user question  
• structured lists of vendors or tools  
• comparison tables summarizing product differences  
• entity coverage across relevant vendors  
• concise explanations that answer the query directly

A page that does not match the structural format expected for the answer may not be selected as a citation even if it performs well in traditional search results.

---

# What Is an AI Citation Cluster

An **AI citation cluster** refers to the group of webpages that AI answer engines repeatedly cite when generating answers for a specific query.

For example, for queries such as “best CRM software”, AI systems often reference a recurring group of comparison pages, vendor guides, and structured product lists.

These recurring sources form the citation cluster for that query.

By analyzing citation clusters, it becomes possible to understand:

• which document structures AI engines prefer  
• which vendors are consistently mentioned  
• which types of pages qualify for citation
